    Educational Metadata

    1. ADEPT Metadata Content Standard Version 1
    The ADEPT Metadata Content Standard Version 1 is a draft document that was discussed and modified (01-09-01 - 01/31/01). It is available for download in two formats.
    • As an Excel spreadsheet. ADEPT Metadata Version 1 Excel file The Excel spreadsheet is the complete content standard.
    • As a Word document. ADEPT Metadata Version 1 Word file The Word document is a short (4 page) one that lists the names of the elements.
    • As a Word document. ADEPT Metadata Controlled Values. This document is a complete listing of all controlled vocabularies considered for the 01/24/01 meeting.
