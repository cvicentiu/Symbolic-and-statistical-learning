Project No: 72

Using Neural Networks in Finance A Practical Approach

    Name: Stephan JOHNSON
    Course: BSc Computing Hons
    Supervisor: PHILLIP BURRELL ( : : )
    Second Assessor: DAVE INMAN (INMANDW : E505 : 7419 )
    Email: JOHNSOS
    Key Areas:
      A.I., Neural Networks

    Summary:

      I propose to perform an investigation into how the current financial indicators can be used in a Neural Network to predict future behavior in the economy. There is strong argument over determining which economic indicators, such as the exchange rates,property,political and military factors, are most responsible for economic fluctuation. One of the key elements of this project will be focused on stock market prediction. Development of a neural prediction system will be biased toward an implementation in C++ mainly because of my familiarity with it as a medium for expressing solutions. I will attempt to justify many of the major issues within the project such as how to determine the profitability of particular portfolios at a given time. Configuration of a neural network involves techniques typically associated with the development of a system ie. appropriate design paradigms whose principles uphold good consistency, enabling reuse & reengineering in other languages. Therefore, sections of the final document will be dedicated to the whole process of training and testing the neural network.
