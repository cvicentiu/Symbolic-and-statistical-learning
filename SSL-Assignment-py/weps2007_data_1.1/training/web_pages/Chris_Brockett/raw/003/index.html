Chris Quirk
   

Chris Quirk is a Researcher in the Natural Language Processing Group.

Background and interests

After studying Computer Science and Mathematics at Carnegie Mellon University, I joined Microsoft in 2000, and started work with the Natural Language Processing group in 2001. Currently I'm working on machine translation, exploring ways to combine traditional linguistic methods with recent statistical advances. In addition, I've spent some time investigating automated MT evaluation and translation confidence scoring. Most recently I've done some investigations into applying effective statistical machine translation techniques to monolingual problems.
