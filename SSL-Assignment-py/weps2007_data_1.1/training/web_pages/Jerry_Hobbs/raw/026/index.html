DARPA Agent Markup Language (DAML)

The goal of the DAML program is to create technologies that will enable software agents to dynamically identify and understand information sources, and to provide interoperability between agents in a semantic manner.

This goal will be pursued by creating an Agent Markup Language built upon XML that allows users to provide machine-readable semantic annotations for specific communities of interest.

These tools embed DAML markup on to web pages and other information sources in a manner that is transparent and beneficial to the users. These tools will be used to build up, instantiate, operate, and test sets of agent-based programs that markup and use DAML.

For more information, visit the DAML home page.
